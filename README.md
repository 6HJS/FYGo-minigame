# 🎴 FYGo - 围棋 × 卡牌策略小游戏

> 基于微信小游戏开发的创新围棋玩法\
> 在传统围棋基础上加入「卡牌技能系统」

------------------------------------------------------------------------

## 🚀 核心特色

### ♟ 围棋规则

-   支持 9x9 / 异形棋盘
-   双人轮流落子
-   气尽提子

### 🧩 卡牌系统

-   🐎 骑兵：持续冲锋
-   💥 自爆兵：范围爆炸 + 摧毁地形
-   🧲 引力：拉动棋子

👉 卡牌为一次性使用

------------------------------------------------------------------------

## 🎮 游戏流程

首页 → 选棋盘 → 选卡牌 → 对局 → 结算

------------------------------------------------------------------------

## 🧱 项目结构

/js /scene /data /utils

------------------------------------------------------------------------

## 🛠 技术

-   Canvas 渲染
-   SceneManager
-   数据驱动设计

------------------------------------------------------------------------

git clone https://github.com/6HJS/FYGo-minigame
